# Suggested Agent Library Assets

本文件在新建 Notion 项目时由 `new-ai-project.sh` 自动填充。

若为空壳，请运行：

```bash
/path/to/ide-toolbox/scripts/query-agent-assets.sh \
  --purpose "{{PROJECT_PURPOSE}}" \
  --type "notion-sync" \
  --privacy "{{PRIVACY_PROFILE}}" \
  --output docs/suggested-assets.md
```

推荐优先查找：

- Notion 双轨项目搭建 SOP
- Notion 日历 / 打卡 / checklist 模式
- manifest / NOTION_INDEX 映射模板
- session-handoff 里程碑收尾模板
